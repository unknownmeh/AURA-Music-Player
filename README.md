
# 🎵 AURA Music Player

**AURA Music Player** is a web-based music player that delivers a relaxing listening experience with **beat-synchronized audio visualization** and a dynamic **audio spectrum display**.

It combines sound and visuals to create an immersive and calming music environment directly in the browser.

---

## ✨ Features

* 🎧 Smooth music playback in browser
* 🌈 Beat-synchronized audio spectrum visualizer
* 🧘 Relaxing visual experience
* ⚡ Lightweight and fast
* 🌐 Fully online — no installation required
* 🆓 Free and open source

---
